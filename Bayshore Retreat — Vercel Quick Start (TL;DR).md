# Bayshore Retreat — Vercel Quick Start (TL;DR)

## 5-Step Deployment

### 1. Create Accounts (5 min)
- GitHub: github.com/signup
- Vercel: vercel.com/signup (use GitHub login)

### 2. Upload Code to GitHub (5 min)
- Create new repo: `bayshore-retreat`
- Upload the exported project files
- (Or ask for help if unsure)

### 3. Deploy to Vercel (2 min)
- In Vercel: "Add New" → "Project"
- Select your GitHub repo
- Click "Deploy"
- Wait 2-3 minutes

### 4. Connect Domain (5 min)
- Vercel shows DNS records
- Add them to Cheapname DNS settings
- Wait 24-48 hours

### 5. You're Live! 🎉
- Your site is now at visitbayshoreretreat.com
- You own everything
- Updates: Edit code → Push to GitHub → Auto-deploys
